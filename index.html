My Portfolio

Here are some projects that I have worked on.

Crystal Collector Game

Crystal Collector Game uses HTML, CSS, Javascript, Bootstrap, and jQuery. The Crystal Game is played by clicking different crystals that are randomly assigned a point value by the Math.random function in javascript.

90's Trivia Game

This is a trivia game that utilizes HTML, CSS, Javascript, Bootstrap, and jQuery. It is a timed trivia game that counts down via intervals. A counter keeps track of how many questions were answered correctly or incorrectly. Results are displayed at the end of the quiz.

GifTastic Search

This project uses HTML, CSS, Javascript, Bootstrap, jQuery and the GIPHY API. The GifTastic application searches the GIPHY API to find gifs by pressing search buttons. There is a selection of buttons that are pre-defined, but users can create a new button by filling a form below.

Train Scheduler App

This app utilizes HTML, Javascript, Bootstrap, Firebase, MomentJS, and jQuery. The Train Scheduler Application begins with a form that requests train information. Once entered, the train is stored in Firebase and placed in the current train schedule. The time data is determined by MomentJS.

Music/Movies App

This is my first group project that utilizes HTML, Materialize, CSS, jQuery, Ajax, Firebase, iTunes API, OMDB API, and TMDB API. This remarkable project will search for your favorite movie and match it with music from the film. The app will also provide alternative choices if your choice is not found.

My Portfolio

Here are some projects that I have worked on. Click on project name to view!

Wireframe

In this project HTML and CSS were utilized to create a skeleton or wireframe design for a webpage. This assignment required css experimentation and accuracy.

Bootstrap Portfolio

This example is a Bootstrap portfolio that utilizes all of the major elements of bootstrap such as column design and navbars. There are no media queries to make it mobile ready however.

Responsive Portfolio

This is a boostrap portfolio that is completely responsive unlike the previous example. This example utilizes media queries to enable resizing for use on any mobile device.

Psychic Game

The goal of this project was to create a game that would run in the browser and feature dynamically updated HTML and CSS powered by our JavaScript code. The player has 9 guesses to find the correct number.

Crystal Collector

The Crystal Collector is a web application that allows users to play a game where they gain points by clicking different crystals. The goal is to match the presented score exactly without going over.

90's Trivia

This is a timed trivia game that counts down via intervals. A counter keeps track of how many questions were answered correctly or incorrectly. Results are displayed at the end.

GifTastic

GifTastic is a web application that utilizes the GIPHY API. The application has the ability to search for gifs based on search criteria. You can also expand your search by adding buttons.

Train Scheduler

The Train Scheduler Application begins with a form that requests train information. Once entered, the train is stored in Firebase and placed in the current train schedule. The time data is determined by MomentJS.

Movie & Music App

This project tries to create a one-stop shop for music and movie information. Our group project will search for your favorite movie and match it with music from the film. The app will also provide alternative choices if your choice is not found.

Liri Node Bot

LIRI Node app is a language and recognition interface. LIRI is a command line node app that takes in parameters and gives you back data. LIRI searches Spotify for songs, Bands in Town for concerts and OMDB for movies.

Bamazon

This project utilizes MySQL & Node to create an Amazon-like storefront app called Bamazon. You are able to look at inventory and prices along with making an order. The database will ring up your order giving you a total.
